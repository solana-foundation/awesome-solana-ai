# Contributing to Awesome Solana AI

Thank you for contributing to this list!

## Guidelines

### Adding a new item

1. Make sure the item is related to AI and Solana development
2. Add your item to the appropriate category
3. Use the format: `[name](link) - Brief description.`
4. Keep descriptions concise (one sentence)
5. Check your spelling and grammar

### Quality standards

- The project should be actively maintained
- Documentation should be available
- The project should provide value to Solana developers

### What NOT to submit

- **No token promotions**: Do not submit projects primarily designed to promote or shill a token. This list is for developer tools and resources, not token launches or investment opportunities.
- **No paid promotions**: Submissions should be genuine recommendations, not paid advertisements.
- **No vaporware**: Projects must have working code and be usable by developers today.

### Pull request process

1. Fork the repository
2. Create a branch for your addition
3. Make your changes to README.md
4. Submit a pull request with a clear description

## Creating a new category

If your item doesn't fit existing categories, you may suggest a new one. Please explain why the new category is needed.

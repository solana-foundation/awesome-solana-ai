# Awesome Solana AI

A curated list of tools, SDKs, and resources for building AI agents on Solana.

## Tools

- **@kevincolahan/sdk** — Compliance audit layer for Solana AI agents. Authorization records, decision audit trails, Merkle-anchored on Solana, CMMC/NIST formatted audit packages, and IRS 2026 tax reporting. Compatible with ElizaOS, Solana Agent Kit, GOAT. [npm](https://npmjs.com/package/@kevincolahan/sdk) | [Docs](https://agentledger-gilt.vercel.app)

## MCP Servers

- **@kevincolahan/mcp** — MCP server for Claude Code agents. Authorization checks, audit trail logging, on-chain anchor verification, and compliance report generation for Solana agent operators. [npm](https://npmjs.com/package/@kevincolahan/mcp)
